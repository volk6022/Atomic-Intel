# Atomic Intel

Сервис автоматизации поиска клиентов, сбора и обогащения данных о компаниях и
мониторинга интернет-ресурсов. Часть экосистемы **Atomic**.

> Публичная прод-версия. Полный функционал распространяется по отдельной договорённости.

## Возможности

- **Лидогенерация и enrichment** — сбор и обогащение карточек организаций.
- **Управляемые вертикали** — стабильный парсинг Яндекс.Карт (карточки, контакты, категории).
- **Автономный research-агент** — LLM-агент с tool-calling: web-поиск (SearXNG) + извлечение
  данных со страниц, собирает структурированные карточки под заданную схему.
- **Мониторинг** — отслеживание изменений на страницах с уведомлениями (Telegram / webhook).

## Архитектура

FastAPI · Taskiq (Redis broker) · Playwright (пул браузеров) · SearXNG (поиск) ·
LLM-эндпоинт (OpenAI-совместимый, в т.ч. локальный).

## Быстрый старт

```bash
cp .env.example .env        # заполнить API_KEY, эндпоинты LLM/Redis/SearXNG
docker compose build
docker compose up -d
curl http://localhost:8000/healthz
```

## API (публичное подмножество)

| Метод | Путь | Назначение |
|---|---|---|
| POST | `/research/run` | Запустить research, вернёт `task_id` |
| GET  | `/research/status/{task_id}` | Статус/результат по токену |
| GET  | `/research/stream/{task_id}` | SSE-стрим прогресса |
| *  | `/yandex-maps`, `/enrichment`, `/monitoring`, `/catalog` | Управляемые вертикали |
| GET  | `/healthz` | Health-чек |

Аутентификация — заголовок `X-API-Key`. Ключи и квоты выдаются через контрол-плейн
(Telegram-бот администрирования).
